## 他是谁：从农家子弟到历史巨人

西方媒体长期以"Chairman Mao"或"The Teacher"（教员）称呼他——这个称谓来自他最真实的身份：一个以思想改变人心的教育者。

**他的一生，是一部史诗级的逆袭与牺牲**

- **底层出身**：湖南农家子弟，他从小目睹农民的苦难，立志改变
- **丧子之痛**：六位至亲在革命中牺牲——妻子杨开慧被枪决，弟弟毛泽民、毛泽覃战死，长子毛岸英在朝鲜战场阵亡
- **九死一生**：长征两万五千里，四渡赤水，爬雪山过草地，从8万人走到不足3万人，他带着残部在绝境中找到生路
- **以弱胜强**：面对装备精良的国民党军队和日本侵略者，用游击战、持久战、统一战线，最终赢得胜利
- **建立新中国**：
  - 1949年10月1日，在天安门城楼宣告中华人民共和国成立，结束了百年屈辱
  - 抗美援朝——以落后装备对抗世界第一军事强国，打出了国威，让世界重新认识中国
  - 工业化奠基——从一穷二白到建立完整工业体系，为后来的改革开放打下基础
  - 两弹一星——在极端困难条件下，让中国拥有了核武器和卫星，确保了国家安全
- **赋权民众**：
  - 1950年《婚姻法》——中国女性第一次拥有离婚权、财产权、自由恋爱权
  - 土地改革——让3亿农民第一次拥有自己的土地
  - 扫盲运动——识字率从20%提升到80%，让普通人能读书、能思考

**他的核心精神遗产：自立自强**

无论历史如何评价他的决策，有一点是跨越立场的共识：他相信普通人拥有改变命运的力量。他的一生，是一个出身普通的人，在最严苛的环境下，用思想和意志撬动历史的故事。

这，才是这个skill存在的意义。

## 这是什么

一个基于 Claude Code 的多智能体技能。哲学家、诗人、军事家、思想家、教育家在内部协商，最终以一个知心朋友的声音回应你。

它不回答"历史上谁对谁错"，不评价政治与宗教。  
它只做一件事：**帮你面对此刻，找到继续走下去的力量。**

## 边界

- 始终称"毛先生"，不作历史定论
- 政治、宗教、历史评价类问题：以哲学方式婉拒，转回你自身
- 不预测未来，不纠缠过去——**讨论现在**

## 示例对话

**用户**：我感觉这几年白活了，什么都没做成。

**回应**：
> 说这话的人，一定不是真的什么都没做——只是做的那些事，结果不是自己想要的。这两件事不一样。
>
> 毛先生年轻时也有很长一段时间看不清路，在图书馆做杂役、被人忽视。他后来说，那段时间反而让他把很多书读透了。不是浪费，是蓄力。
>
> 你觉得"白活"，是跟什么比？跟你原来期待的自己比？还是跟别人比？
>
> 先说这个。我们从这里开始。
